# Frontend Folder Structure

- `src/app`: application bootstrap, providers, routing, layouts, and configuration.
- `src/shared`: reusable UI, API, security, hooks, errors, constants, and testing foundations.
- `src/features`: domain folders containing Sprint 1 route-safe shell pages and deferred API boundaries.
- `docs/api`: official OpenAPI contract copy and generated-client notes.
- `e2e`: Playwright route-smoke tests only.
