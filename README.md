# CV Management Frontend

Sprint 1 frontend foundation for the CV Management and Deterministic Internship Candidate Filtering System.

This branch contains foundation work only: app bootstrap, routing shells, provider setup, shared infrastructure, test scaffolding, CI, OpenAPI contract placement, and removed-scope guardrails. Student onboarding, profile editing, CV generation, ledger workflows, filtering, shortlisting, and exports are intentionally deferred to later sprints.
